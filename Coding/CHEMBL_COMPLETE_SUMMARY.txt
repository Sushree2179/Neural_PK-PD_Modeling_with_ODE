🔗 API ACCESS FOR REAL DATA
=============================

When ready to upgrade to official ChEMBL data:

Install: pip install chembl-webresource-client

Python API:
  from chembl_webresource_client.new_client import new_client
  activity = new_client.activity
  results = activity.filter(
    target_chembl_id='CHEMBL231',
    standard_type__in=['Ki','IC50'],
    pchembl_value__gte=5.0
  )

REST API:
  https://www.ebi.ac.uk/chembl/api/data/activity.json?
  target_chembl_id=CHEMBL231&standard_type__in=Ki,IC50

Full Database:
  https://ftp.ebi.ac.uk/pub/databases/chembl/ (5.2 GB SQLite)
